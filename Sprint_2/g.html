Cash-Flow

Salary & Expense Tracker — Video Script

⏱ Duration: ~2 min 45 sec 🌐 Language: English 🛠 Stack: HTML · CSS · JavaScript 🏢 Prodesk IT — Sprint 2
0:00 – 0:20 🎤 Introduction
📺 Screen: Browser open — full homepage visible
"Hello everyone! Today I'm going to walk you through my project — Cash-Flow — a smart Salary and Expense Tracker built entirely using pure HTML, CSS, and JavaScript. No frameworks, no backend — just clean, modern frontend web development."
0:20 – 0:40 🖥️ UI Overview
📺 Screen: Slowly scroll the page from top to bottom
"First, let's look at the UI. The app features a dark theme with a glassmorphism sticky header, smooth animations, and premium Google Fonts — Inter and Outfit.

At the top, there are three summary cards: Total Salary, Total Expenses, and Remaining Balance — all of which update in real-time whenever any data changes."
0:40 – 1:00 💼 Setting Salary
📺 Screen: Type salary → click "Set Salary" → show pop animation → show validation error
"On the left side, we have the Set Your Salary card. I'll enter my monthly salary here and click 'Set Salary'.

Notice how the salary card at the top updates instantly with a pop animation. The form also has real-time input validation — if you enter an empty or negative value, a descriptive error message appears automatically."
1:00 – 1:25 ➕ Adding Expenses
📺 Screen: Add 4 expenses — Rent, Groceries, Netflix, Petrol
"Now let's add some expenses using the Add Expense form. I'll add Rent, Groceries, Netflix, and Petrol.

Notice that each expense is automatically assigned a smart emoji icon based on its name — 🏠 for Rent, 🛒 for Groceries, 📺 for Netflix, ⛽ for Petrol. This is implemented using keyword-matching logic in JavaScript.

Each new entry appears with a smooth slide-in animation, and the Expenses and Balance cards update in real-time."
1:25 – 1:40 📊 Donut Chart
📺 Screen: Show chart animating with each expense, hover over it
"On the right side there's a Spending Overview donut chart built using the Chart.js library.

It updates in real-time — red represents Total Expenses, and purple represents Remaining Balance. Hovering over a segment shows the exact amount and its percentage of your total salary."
1:40 – 1:55 ⚠️ Critical Balance Alert
📺 Screen: Add expenses until balance drops below 10% of salary
"Here's an important safety feature — if the remaining balance drops below 10% of the salary, a red warning banner slides in from the top of the screen, displaying exactly how much percentage is left.

The Balance card also turns red and shows a danger pulse animation. The banner can be dismissed using the close button."
1:55 – 2:15 🌐 Live Currency Toggle (INR ↔ USD)
📺 Screen: Click "Switch to USD" → spinner → rate loads → all values convert to USD
"In the header there's a 'Switch to USD' button. When clicked, the app makes a live API call to fetch the real-time USD to INR exchange rate.

It uses two APIs with automatic fallback:
① Frankfurter API (primary)
② Open Exchange Rate API (fallback if first fails)

A loading spinner shows while fetching, and once the rate is received, all amounts — salary, expenses, balance — are instantly converted to USD. Switching back to INR is one click."

Primary API: https://api.frankfurter.app/latest?from=USD&to=INR

Fallback API: https://open.er-api.com/v6/latest/USD

2:15 – 2:30 📄 PDF Report Download
📺 Screen: Click "Download Report" → PDF downloads → open and show contents
"The 'Download Report' button in the header generates a professional A4 PDF report using the jsPDF library.

The PDF includes a styled header, a summary section with Salary, Expenses, and Balance, a complete expense table with dates, and the Net Remaining Balance at the bottom. If USD mode is active, it also includes a currency conversion note."
2:30 – 2:45 💾 localStorage & Responsive Design
📺 Screen: Refresh page — data persists. Resize to mobile — layout stacks
"All data is saved to the browser's localStorage — so even after refreshing the page, your salary and expenses are still there.

The app is also fully responsive — on smaller screens, the two-column layout automatically stacks into a single column, making it usable on mobile as well."
2:45 – 3:00 🏁 Closing
📺 Screen: Full project running smoothly — all features visible
"So that was Cash-Flow — a fully functional Salary and Expense Tracker built with pure HTML, CSS, and JavaScript. Thank you!"
