# 💸 Cash-Flow — Salary & Expense Tracker

> Sprint 02 | Vanilla JS + HTML + CSS

A simple dashboard to set your salary, track expenses, and watch your balance update live — no framework, no server.

🌐 **Live:** [prodesksprint2.vercel.app](https://prodesksprint2.vercel.app/)

---

## Features

- Set salary + add expenses (live balance calculation)
- LocalStorage — data persists after browser reload
- Donut Chart — salary vs expenses visualization
- Delete expense with slide-out animation
- One-click PDF Report download
- Live INR ↔ USD conversion (Frankfurter API)
- Warning banner when balance drops below 10% of salary

---

## Tech Used

`HTML` · `CSS` · `Vanilla JS` · `Chart.js` · `jsPDF` · `Frankfurter API` · `LocalStorage`

---

## How to Run

```
index.html → double-click → opens in browser ✅
```
